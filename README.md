# WM Client Intake & Audit Dashboard

A structured intake and audit tool built for CRM/ESP agency projects.

Designed to standardise how a marketing agency onboards new clients 
onto platforms like Customer.io, Klaviyo, Braze, and others.

## What it does
- Guides the team through a consistent week-1 data audit
- Tracks workspace health: profiles, campaigns, attributes, events, data sources, automations
- Scores project risk across 8 factors
- Maintains a living checklist with owner and status tracking
- Generates a client-ready weekly update email in one click
- Produces a clean client summary for PDF export

## Built with
HTML · CSS · Vanilla JavaScript · No dependencies
